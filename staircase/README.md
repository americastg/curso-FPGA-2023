# Problema: Escada

## Descrição
Dado um número inteiro n que representa a altura da escada, o objetivo é imprimir uma escada em forma de "#". Cada degrau deve ter tamanho 1 e largura n.

## Exemplo de entrada e saída


    Entrada: 6
    Saída:
    
    ```shell
         #
        ##
       ###
      ####
     #####
    ######
    ```

### Observações

    n deve ser >= 1 e < 16.
